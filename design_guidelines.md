# Design Guidelines for سليمان Luxury Agency Website

## Design Philosophy
**Ultra-premium, cinematic Arabic RTL experience combining:**
- Apple.com smoothness + Lamborghini luxury + Black×Gold royal theme
- Mobile-first responsive design
- Glass morphism, soft shadows, cinematic depth

## Color System
- **Deep Black:** #0A0A0A (primary background)
- **Carbon Black:** #101010 (secondary surfaces)
- **Pure White:** #FFFFFF (text, contrast)
- **Royal Gold:** #F2C400 (primary accent, CTAs, icons, glows)

## Typography
- **Primary Font:** Tajawal or Cairo (Arabic premium sans-serif)
- Mobile-first responsive sizing
- RTL text alignment throughout

## Spacing System
Consistent spacing scale: **8px / 16px / 24px**
- Use these units for padding, margins, gaps
- Maintain rhythm across all sections

## Layout Components

### Header (Sticky)
- Transparent initially with blur-on-scroll effect
- Logo (provided asset) on right side (RTL)
- Navigation links with gold hover states
- Mobile hamburger menu with smooth gold animations

### Hero Section (Cinematic)
- **Right side:** 3D floating mockup with soft gold shadow and parallax motion
- **Left side:** 
  - Eyebrow text: "وكالة إبداع رقمية"
  - Large hero title
  - Descriptive subtitle
  - Two CTAs: Primary gold button + secondary outline
- Gold particle animation background
- Parallax mouse layers
- Full viewport height

### Signature Stats (4 Cards)
- Gold icons
- Large numbers
- Shadow elevation with hover glow effects
- Responsive grid (1 column mobile → 4 columns desktop)

### Services Section (8 Cards)
- Section title + subtitle
- Glass morphism card effects
- Gold icons
- Service title + description
- "اعرف المزيد" links
- 3D hover transforms
- Grid: 1 col mobile → 2 tablet → 4 desktop

### Portfolio Grid (10 Projects)
- Large project images
- Hover zoom effects
- Overlay with title + category
- Modal detail views with screenshots and descriptions
- Masonry or standard grid layout

### Client Logos Marquee
- Full-width infinite RTL scroll
- 12-16 client logos
- Grayscale default → gold on hover
- Seamless loop animation

### Featured Stores (6 Examples)
- Premium store mockup images
- Gold borders
- Category labels
- Short descriptions
- Hover elevation effects

### Testimonials (8 Reviews)
- Customer name + review text
- 5-star gold rating display
- Fade-up scroll animations
- Grid layout: 1 col mobile → 2 tablet → 4 desktop

### CTA Strip
- Full-width gold gradient background
- Large heading text
- Primary gold button
- High contrast white text

### Footer (Ultra Premium)
- Logo (provided asset) placement
- Gold divider line
- 4 columns: About / Quick Links / Contact / Mini Form
- Social icons with gold glow
- Dark gold gradient background

## Store Page
- Product grid with cards
- Product image + name + price (SAR)
- Quantity selector
- PayPal "Buy Now" button per product
- Product preview modal
- Grid: 1 col mobile → 2 tablet → 3 desktop

## Contact Page
- Split layout: Left (info) + Right (form)
- Premium form styling with gold accents
- Gold glow submit button
- Location, phone, email details

## Animation & Effects
- **Parallax:** Hero mockup, background layers
- **Gold particles:** Floating in hero background
- **Scroll animations:** Fade-up on sections
- **Hover effects:** 3D card transforms, glow, elevation
- **Smooth scroll:** Native smooth behavior
- **Page transitions:** Cinematic fades

## Component Styling
- **Rounded corners:** 12px to 18px
- **Shadows:** Soft, layered for depth
- **Glass panels:** Backdrop blur with transparency
- **Gold glow:** box-shadow with #F2C400
- **Borders:** Subtle gold accents where appropriate

## Images
- **Hero:** 3D floating mockup/device showcase (right side in RTL)
- **Portfolio:** 10 high-quality project screenshots
- **Featured Stores:** 6 luxury store mockups
- **Products:** 8 product/service images
- **Client Logos:** 12-16 brand logos

## Responsive Breakpoints
- **Mobile:** < 768px (1 column, stacked layout)
- **Tablet:** 768px - 1024px (2 columns)
- **Desktop:** > 1024px (3-4 columns, full features)

## Critical Requirements
- RTL layout throughout
- Mobile-first design approach
- Lazy-loading images
- SEO meta tags + Open Graph
- Consistent gold accent usage
- Premium feel on all devices
- Smooth performance with animations
